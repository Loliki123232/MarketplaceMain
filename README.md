# 🛍️ Маркетплейс - WPF приложение для управления интернет-магазином

[![.NET](https://img.shields.io/badge/.NET-8.0-blue.svg)](https://dotnet.microsoft.com/)



## 📋 О проекте

**Маркетплейс** — это десктопное приложение для управления интернет-магазином, разработанное на платформе WPF (.NET 8.0). Приложение позволяет просматривать товары, добавлять их в корзину, оформлять заказы, управлять каталогом товаров и просматривать историю заказов.

## 🎯 Основные возможности
### Модуль 1: Основной функционал

| Функция | Описание |
|---------|----------|
| **📦 Каталог товаров** | Просмотр всех товаров с фильтрацией по категориям и поиском |
| **🛒 Корзина** | Добавление товаров, изменение количества, удаление |
| **📋 История заказов** | Просмотр оформленных заказов с детальной информацией |
| **➕ Добавление товара** | Добавление новых товаров в каталог |
| **🧾 Чек заказа** | Формирование текстового чека после оформления заказа |
| **💰 Скидки** | Автоматический расчёт скидок в зависимости от суммы заказа |

### Модуль 2: Возврат товара

| Функция | Описание |
|---------|----------|
| **🔄 Оформление возврата** | Создание заявки на возврат товара из карточки заказа |
| **✅ Проверка возможности возврата** | Валидация статуса заказа и срока возврата  |
| **📝 Форма создания заявки** | Указание причины возврата и количества товара |
| **📊 История возвратов** | Просмотр всех заявок с цветовой индикацией статуса |
| **🔍 Просмотр причины** | Детальный просмотр причины возврата |
| **🔄 Восстановление склада** | Автоматическое восстановление товара на складе при одобрении возврата |

### Модуль 3: Промокоды и купоны

| Функция | Описание |
|---------|----------|
| **🎫 Применение промокода** | Ввод промокода в корзине для получения скидки |
| **✅ Проверка действительности** | Валидация промокода по датам, активности и минимальной сумме |
| **📊 Типы скидок** | Поддержка процентной и фиксированной скидки |
| **🚫 Запрет повторного использования** | Контроль использования промокода одним пользователем |
| **📈 Отображение скидки** | Показ суммы скидки по промокоду в корзине |
| **📝 Сообщения о результате** | Информирование об успешном применении или ошибке |
